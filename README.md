# Gestor de Tareas con Ramas

Este proyecto es un gestor de tareas desarrollado para facilitar la organización y seguimiento de tareas utilizando la metodología de 'feature-branching'. Cada tarea se puede asociar a una rama específica en el repositorio, permitiendo que diferentes características y mejoras se desarrollen en paralelo sin interferir unas con otras.

## Características
- **Creación de Tareas**: Crea nuevas tareas con descripciones y prioridades.
- **Gestión de Ramas**: Asocia nuevas características a ramas específicas, permitiendo que los desarrolladores trabajen en paralelo.
- **Seguimiento de Progreso**: Actualiza el estado de las tareas y visualiza el progreso de cada rama.
- **Interfaz Amigable**: Una UI Javascript simple que permite una fácil navegación y gestión de tareas.

## Instalación
1. Clona el repositorio.
2. Ejecuta `npm install` para instalar las dependencias.
3. Inicia la aplicación con `npm start`.

## Contribuciones
Las contribuciones son bienvenidas. Por favor, sigue el flujo de trabajo de 'feature-branching' al agregar nuevas características.
